# supabase/

Supabase project files for SQEPify (a separate project from the Control Room).

- `baseline/SCHEMA_BASELINE.sql` — the schema snapshot, the reference of record.
  Regenerated by re-extraction, not hand-edited per change.
- `baseline/EXTRACTION_QUERIES.sql` — the queries that regenerate the baseline.

Edge functions are added here in later build-order steps (Control Room sync, then the
AI functions). They are deployed by pasting into the Supabase dashboard.

How database changes are made and recorded is governed by `docs/DATABASE_WORKFLOW.md`.
The running record of changes is `CHANGELOG.md` at the repo root.
